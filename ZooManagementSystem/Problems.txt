Initial Design Smell:
1. Class Dog gak masuk ke Carnivorous Animals (Missing Abstraction & Incorrect Hierarchy)
    - Dog punya banyak atribut sama kayak Carnivorous
    - Dog juga makannya daging
    - harusnya masuk CarnivorousAnimal intinya
    --> bisa nambahin possible new design smell (point 1 additional design smell)

    Possible Solution : 
    -   Carnivorous jadi interface aja soalnya ikan dan penguin juga bisa carnivorous
    -   Tambahin abstract class Animals untuk jadi awalan
    -   Ubah CarnivorousAnimals jadi Mammal
    -   Ubah AquaticAnimals jadi Fish
    -   Tambahin Abstract class Bird untuk penguin

2. Class Zoo handle terlalu banyak responsibilities (Multifaceted Abstraction || Large Class)
    Responsibilities:
    -   ngatur Database semua hewan + CRUD (Lion, Penguin, Tiger, dll)
    -   print semua hewan
    -   mengecek height penguin
    -   feed hewan
    -   nambahin umur semua hewan setaun
    -   listen to all animals in the Zoo
    -   mencari most popular fish colour
    Possible Solution :

3. Class Main terlalu banyak responsibilities (Multifaceted Abstraction || Large Class)

4. Hierarchy utama abstract class "Animals" tidak terdapat (missing hierarchy)
   Hal ini dideteksi dari adanya Data Clumps di hampir seluruh class
   Solusi : di Extract SuperClass baru (Animal)
    - Semua class hewan memiliki atribut dan function yang sama seperti:
        - age
        - happiness
        - Lifespan
        - Noise
        - getter setter dari atribut di atas
        - feed()
        - toString()
        - ageOneYear()
        - makeNoise()

5. Atribut tambahan di class yang extend carnivorous memiliki atribut sama
sebaiknya di move attribute ke superclassnya.

6. Beberapa hall seperti noise dan MeatCalcFemale/MeatCalcMale sebaiknya menjadi behavior saja bukan attribute

7. Multifaceted Abstraction and Refused Bequest di AquariumFish
    - method seperti randomizeColour dan randomizePattern bukan responsibility AquariumFish
    - Subclass yang extend dari AquariumFish tidak 

Tahapan Benerin:
    - bikin Abstract Class Animal buat top Abstraction (Design Smell-3)
    - correspond semua class di bawahnya dengan Hierarchy baru
        - rename AquariumFish jadi Fish saja
        - CarnivorousAnimal, Fish, dan Penguin extends Animal
        - Dog masuk ke CarnivorousAnimal
    - Perbaiki SRP untuk class Fish
        - di dalam class Fish terdapat method randomizeColour dan randomizePattern
        - buat Class FishAttributeRandomizer untuk handle responsibility tersebut
    - Perbaiki Zoo class
        - di dalam Zoo Class terdapat banyak sekali responsibilities, menjadikan class itu hampir God Class
        - Zoo Class sebaiknya memiliki responsibility untuk menghandle request dari Main saja
            - lakukan Extract Class, untuk semua method yang berhubungan dengan Fish akan ditaruh di class FishService
                - addNewFish
                - addRandomFish
                - getAllFish
            - lakukan yang sama untuk setiap hewan lainnya juga (tambahin setiap add dan get di Service)
            - Print logic bikin di ZooPrinter
        - Zoo Class tetap memiliki responsibility untuk
            - add, feed, dan lain-lain namun hanya sebatas sebagai tampilan saja
            - listenToAll, feedAll, ageOneYearAll, dll
